# ADR-0001: Use Raspberry Pi OS Lite (64-bit)

- Status: Accepted
- Date: 2026-07-10

## Decision

Use Raspberry Pi OS Lite (64-bit) for `cycling-prod`.

## Rationale

The production host is headless. A desktop environment is unnecessary and would add maintenance and resource overhead. The Lite image provides the required ARM64 base with a smaller footprint and reduced attack surface.
